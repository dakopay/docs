Install the SDK in order to use our protocol:

## Install SDK

```jsx

// instal sdk
npm i projectrepa/SDK

```

## Usage SDK

```jsx

// Use
import { setChain, ABI, checkAddress, getNetwork } from 'repa-sdk';

// Set Network Chain
setChain(5);

// call functions
var addr = await checkAddress(ethaddress);

console.log(addr);

```
